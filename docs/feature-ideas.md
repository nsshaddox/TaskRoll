# Feature Ideas

A reference list of features found across task management and random task apps. Not all of these are planned for implementation — this is a catalog of possibilities.

## Core Task Management
- Create tasks (title, description, notes)
- Edit tasks
- Delete tasks (single, bulk)
- Duplicate tasks
- Archive tasks
- Mark complete / incomplete
- Reorder tasks (drag and drop)
- Undo / redo actions

## Organization & Structure
- Categories / folders / projects
- Tags / labels
- Subtasks / checklists
- Task dependencies
- Nested task hierarchies
- Sections within lists
- Multiple lists / boards
- Inbox / default list

## Task Metadata
- Priority levels (low, medium, high, urgent)
- Due dates
- Due times
- Start dates
- Estimated duration
- Actual time tracked
- Creation date
- Last modified date
- Notes / rich text body
- File attachments
- Links / URLs
- Color coding

## Filtering & Searching
- Search by keyword
- Filter by tag
- Filter by priority
- Filter by due date
- Filter by status (active, completed, archived)
- Filter by category
- Sort by various fields
- Saved filters / smart lists

## Views
- List view
- Board / Kanban view
- Calendar view
- Gantt / timeline view
- Grid / table view
- Today / upcoming view
- Completed tasks view
- Agenda view

## Scheduling & Time
- Reminders / notifications
- Recurring tasks (daily, weekly, monthly, custom)
- Time blocking
- Snooze reminders
- Overdue indicators
- Deadline warnings

## Random Task Selection
- Pick random task (from all)
- Pick random task (from filtered subset)
- Pick random task by priority weight
- Pick random task by tag/category
- Pick random task by estimated duration
- Pick random task excluding recent picks
- Spin wheel animation
- Reveal animation
- History of random selections
- "Roll again" / skip
- Lock in / confirm selection
- Reject and re-roll

## Gamification
- Streaks (completing tasks daily)
- Points / XP system
- Achievements / badges
- Level progression
- Completion statistics
- Productivity graphs
- Confetti / celebration animations
- Challenge mode (complete N random tasks)

## Completion Flow
- Mark done with swipe gesture
- Undo completion
- Completion timestamp
- "Skip" option (don't mark done, move on)
- "Later" / snooze option
- Post-completion note / reflection

## Collaboration
- Share lists with others
- Assign tasks to people
- Comments / discussion on tasks
- @mentions
- Activity feed / change history
- Real-time sync across members

## Sync & Storage
- Local-only storage (offline first)
- Cloud sync
- Multi-device sync
- Conflict resolution
- Backup / export
- Import from other apps

## Import / Export
- Export to CSV / JSON / plain text
- Import from CSV
- Share task as text
- Print task list
- Copy task details to clipboard

## Notifications & Reminders
- Push notifications
- In-app notifications
- Location-based reminders
- Notification snooze
- Daily digest summary
- Quiet hours

## Settings & Customization
- Theme (light, dark, system)
- Accent colors
- Font size
- Default sort order
- Default due time
- Week start day
- Badge count settings
- Widget configuration

## Widgets & Integrations
- Home screen widget (task list)
- Home screen widget (random pick button)
- Calendar integration
- Voice input (Siri, Google Assistant)
- Shortcuts / quick actions
- Share extension (add task from other apps)
- Notification action buttons

## Accessibility
- Screen reader support
- Dynamic text size
- High contrast mode
- Keyboard navigation
- Haptic feedback

## Onboarding & Help
- Guided tutorial
- Sample tasks
- Tips / hints
- Keyboard shortcuts reference
- Empty state illustrations
